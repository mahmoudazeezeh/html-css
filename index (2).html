السمات المستخدمة

السمة الاختصار مثال الاستخدام
src source
تحدید مصدر الصورة أو موقعھا
<img src="image/bird.jpg" />
alt alternative
تحدید بدیل نصي عن الصورة
<img src="image/bird.jpg" alt="bird" />
width width
تحدید عرض الفراغ المخصص لل tag
<p width="200px"> knowledge</p>
height height
تحدید ارتفاع الفراغ المخصص لل tag
<p height="250px"> knowledge</p>
href Hyper reference
تحدید العنوان الخاص بال link
<a href="contact"> Contact page</a>
target ---
لتحدید طریقة فتح الرابط
<a href="contact" target="blank"> Contact page</a>
type ---
لتحدید نمط الترقیم للقوائم
<ul type="a"> <li> desc </li> </ul>
start ---
لتحدید نقطة البدایة في ترقیم القوائم
<ul type="a" start="3"> <li> desc </li> </ul>
id identity
تحدید اسم خاص بال tag
<p id="contact"> Contact page</p>
border ---
لإضافة سماكة لحدود الجدول
<table border='1'> </table>
colspan Column span
لدمج الخلایا في نفس الصف
<td colspan='2'> </td>
rowspan Row span
لدمج الخلایا في نفس العمود
<td rowspan='3'> </td>
type ---
لتحدید نوع البیانات
<input type="text" />
for ---
لجعل ال label یشیر الى input
<label for="name"> </label>
placeholder ---
لإدراج إرشاد داخل ال input, عن طبیعة البیانات التي یجب ادخالھا
<input placeholer="First Name" />
name ---
لإعطاء اسم لل input, ینتظره ال backend
<input name="password" />
value ---
لإعطاء قیمة محددة لل input
<option value="female"> </option>
required ---
لجعل الادخال إلزامیا في ال input
<input required />

خصائص CSS - الجزء الأول

الخاصية القيمة الافتراضية الخيارات مثال الاستخدام
color black Any color
للتحكم بلون الخط
color: red;
background-color white Any color
للتحكم بلون الخلفية
background-color: purple;
width Depend on tag px or % or fit-content
للتحكم بالعرض المخصص للعنصر
width:20%;
height auto auto or px or %
للتحكم بالارتفاع المخصص للعنصر
height:400px;
text-align left Left, right or center
للتحكم بمحاذاة النص
text-align:center;
display Depend on tag Inline,block, inline-block,none
للتحكم في المساحة التي يحجزها العنصر في السطر
display:inline-block;
float --- Left, right
لترتيب العناصر بجانب بعضها
float: left;
clear --- Left, right, both
لحل مشكلة clrfix مع float
clear:both;
font-size Depend on tag Any number
للتحكم بحجم الخط
font-size:22px;
font-weight Usually 400 Any number
للتحكم بسماكة الخط
font-weight:600;
font-style normal Normal, italic
للتحكم بنمط استقامة الخط
font-style:italic;
font-family --- Any font name
للتحكم بعائلة الخط
font-family:Roboto;
text-transformation lowercase Lowercase,uppercase, capetalize
للتحكم بالحروف
text-transfomation: uppercase;
text-decoration-line none None,overline,underline, line-throught
للتحكم بمكان الخط المرافق للنص
text-decoration-line: line-through;
text-decoration-style solid Solid,dotted,dashed,etc…
للتحكم بنوع الخط المرافق للنص
text-decoration-style: dashed;
text-decoration-color --- Any color
للتحكم بلون الخط المرافق للنص
text-decoration-color: red;
text-decoration --- ---
للتحكم بخصائص الخط المرافق للنص
text-decoration: red solid overline;
letter-spacing Depend on font family Any number
للتحكم بالمسافة بين الحروف في الكلمة
letter-spacing:20px;
word-spacing Depend on font family Any number
للتحكم بالمسافة الافقية بين الكلمات
word-spacing:12px;
line-height --- Any number
للتحكم بارتفاع السطر
line-height:30px;

خصائص CSS - الجزء الثاني

الخاصية القيمة الافتراضية الخيارات مثال الاستخدام
direction left ltr, rtl
للتحكم باتجاه اضافة العنصر

margin (shorthand) Depend on tag Any number or auto
للتحكم بقيمة الهوامش بين العناصر

padding (shorthand) Depend on tag Any number
للتحكم بقيمة الهوامش الداخلية
padding:auto;
border (shorthand) Depend on tag Any number with any color with any style
للتحكم بحدود العنصر
border:5px red solid;
box-sizing content-box content-box, border-box
للتحكم بالتزام العنصر بحدود الاب
box-sizing:border-box;
overflow visible visible,hidden,scroll,auto
للتحكم بظهور المحتوى الزائد عن ابعاد العنصر
overflow:hidden;
position static absolute,relative,fixed,sticky
للتحكم باتجاه اضافة العنصر
position:absolute;
top --- Any number
للتحكم بالمسافة بين العنصر والاعلى
top:20px;
bottom --- Any number
للتحكم بالمسافة بين العنصر والاسفل
bottom:20px;
right --- Any number
للتحكم بالمسافة بين العنصر واليمين
right:20px;
left --- Any number
للتحكم بالمسافة بين العنصر واليسار
left:20px;
z-index 0 any integer
للتحكم باولوية ظهور العنصر
z-index:-1;
border-radius --- Any number
للتحكم في انحناء اطراف الحدود
border-radius:20px;
visibility visible visible,hidden
للتحكم بظهور العنصر
visibility:hidden;
opacity 1 0-1
للتحكم بشفافية العنصر
opacity:.35;
content --- Any text
لكتابة محتوى ل before و after
content: 'hi baby';
cursor auto Many values according to the use
للتحكم بشكل المؤشر
cursor:pointer;
background-image --- images
لاضافة صورة كخلفية
background-image:url();
box-shadow --- ---
لاضافة ظل للعنصر
box-shadow: 10px 10px 10px 10px red;
text-shadow --- ---
لاضافة ظل للنص
text-shadow: 10px 10px blue;
animation --- ---
لاضافة حركات تلقائية
animation: play 3s 1s infinite;
transition --- ---
للتحكم بمدة تنفيذ الخاصية
transition:opacity .5s;
transform --- ---
للتحكم بحركة العنصر
transform:rotate(50deg);
